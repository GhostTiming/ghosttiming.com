Step 1: Cellular Network Connect
Turn on the cellular adapter first and confirm the Pro2 actually has cellular service before doing anything with server connection.
Tap to toggle
Goal
Get the green status bubble on the cellular adapter. Do not jump ahead to server connection until that part is good.

Cellular connection steps

  1. Go to Settings → Network Adapters.
  2. Tap Cellular → Edit.
  3. Make sure Power is set to On.
  4. Wait 10 to 20 seconds and look for the green status bubble.
  5. If Power is On and you still do not see the green bubble after 1 to 2 minutes, toggle Power Off → On and try again.
  6. This can take a couple tries. Do not start messing with the server settings yet.

Step 2: Time Setup and Sync With Mini
Set the Pro2 time using NTP first, use manual time only as backup, then broadcast from the running clock to sync with the Mini.
Tap to toggle
Priority order
  • NTP Time is the preferred method.
  • Manual Time is backup only and should not be used casually.
  • After time is set, press the running clock and use Broadcast to sync with the Mini.
  • If you press Broadcast and hear no beep, go to Mini LAN Fix. That means the boxes are not reaching each other.

Preferred method: NTP Time

  1. From the main ChronOS screen, tap the running time display.
  2. Go to Race Ops → Time.
  3. Tap Time Sync → select NTP Time.
  4. Toggle Auto Sync Time to On.
  5. Confirm the Pro2 time looks right compared with your phone.

Fallback only: Manual Time

Do not press Set Time without direction from the timer.

  1. Tap the running time display → Race Ops → Time.
  2. Tap Manual Time → Config Time.
  3. Set the date → Next.
  4. Set hour, minute, second and AM/PM → Update.
  5. Only when instructed, press Set Time.

Sync the Mini after time setup

  1. Return to the main screen and tap the running clock.
  2. Press Broadcast to sync the time with the Mini.
  3. Listen for the Mini to beep.
  4. If you hear the beep, the boxes are talking and the sync should be good.
  5. If you do not hear a beep, go straight to Mini LAN Fix.

Step 3: Set Event and Point Name
Set the event name from the crew email and make sure the point name matches the location.
Tap to toggle
Important
Changing the Event Name creates a new session of data. Do not change it while athletes are actively crossing unless you intentionally want a new session.

Event Name rules

  • Use the value from the crew email.
  • Use ALL CAPS.
  • Use NO SPACES.

Event Name steps

  1. From the main ChronOS screen, tap the Event Name in the top left.
  2. On the Controller screen, tap the Event Name field to edit it.
  3. If needed, tap the X next to the field to clear it quickly.
  4. Type the new Event Name exactly as provided in the crew email.
  5. Press Save.

Point Name standards

  • START = Start line only
  • FINISH = Finish line only
  • MAIN = Start and finish are the same physical location
  • ANNOUNCER = Announcer line system
  • For other locations, use an intuitive name like MILE5. If unsure, reach out to the timer.

Point Name steps

  1. On the Controller screen, tap the Point Name field.
  2. Use the ellipsis (...) button to pick a standard name when applicable.
  3. If it is a split or custom location, type an intuitive Point Name and press Save.

Mini LAN Fix
Use this when you press Broadcast from the running clock and the Mini does not beep, which means the boxes cannot reach each other.
Tap to toggle
Hard rule
The Pro2 LAN IP and the Mini LAN IP must never be the same.

Pro2 LAN settings

  • IP format: 172.20.23.x
  • x must be a multiple of 5 between 100 and 250
  • Netmask: 255.255.252.0
  • Gateway: 172.20.23.1

Mini LAN settings

  • IP format: 172.20.23.x
  • x range: 50 to 99
  • Netmask: 255.255.252.0
  • Gateway: 172.20.23.1

Fix on Pro2 (ChronOS)

  1. Go to Settings → Network Adapters.
  2. Select LAN → tap Edit.
  3. Set the Pro2 IP to 172.20.23.x where x is a multiple of 5 between 100 and 250.
  4. Set Netmask to 255.255.252.0.
  5. Set Gateway to 172.20.23.1.
  6. Press Save or Apply.

Fix on Mini

  1. Open the Mini menu → go to Network Cfg.
  2. Set Ctrlr IP to 172.20.23.x where x is between 50 and 99.
  3. Set Ctrlr Netmsk to 255.255.252.0.
  4. Set Ctrlr Gatewy to 172.20.23.1.
  5. Confirm changes when prompted.
Then retry the sync
Go back to the Pro2, tap the running clock and press Broadcast again. If it still does not beep, call the timer.

Mini Track Clear Message (AKA Make the Beeping Stop)
Use this when the MiniTrack keeps beeping. Read the message first, then acknowledge or clear it.
Tap to toggle
Always do this first
  1. Press Menu.
  2. Press 1 then 1 to open Messages.
  3. Read the message.
  4. Acknowledge or Clear the message to stop the alert.

Common message: Time mismatch

  • After clearing the message, confirm whether the Pro2 time and Mini time still match.
  • If they still match, disregard and continue.
  • If they significantly mismatch, call the timer.

Common message: Power or battery

  • If the message mentions power or battery, put a fresh battery in the slot that does not have the in-use light.
  • If it says the DC cable is disconnected, make sure the cable is snug in the port and connected to a power source that is on.
  • Then clear the message again using Menu → 1 → 1.
  • If time does not significantly mismatch and it is only a power message, you should be all set after clearing it.
  • For all other errors, or any significant time mismatch, call the timer.

Emergency - Wi‑Fi Server Connect
Use this only if cellular will not work. Join Wi‑Fi first, then connect to the server through a Wi‑Fi server profile.
Tap to toggle
Emergency only
This is the backup plan when cellular refuses to cooperate.

Part A — Join Wi‑Fi

  1. Make sure the router or hotspot is On and broadcasting Wi‑Fi.
  2. Go to Settings → Network Adapters → select WIFI → Edit.
  3. Toggle Wi‑Fi Power On if needed.
  4. Select the network → press Connect.
  5. Enter the password → press Save → then press Join.
  6. Wait 10 to 30 seconds. Status should show Connected and the Wi‑Fi icon should not have a slash.

Part B — Connect to the Server

  1. Go to Race Ops → Server Connection.
  2. Press + to add a server entry if needed.
  3. Choose WIFI type, enter a name and enter the server IP: 50.17.233.218.
  4. Press Add.
  5. Tap the new WiFi server entry → press Connect.
  6. Confirm it changes to Connected and the cloud icon has no slash.
