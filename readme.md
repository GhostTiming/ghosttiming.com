# Ghost Timing Website

Lead-focused landing website for Ghost Timing & Race Support.

## Files

- `index.html` - Main page and content structure
- `styles.css` - Visual design and responsive layout
- `script.js` - Placeholder custom lead-form behavior
- `assets/logo-white.png` - White logo version
- `assets/logo-black.png` - Black logo version

## Local Preview

Open `index.html` directly in your browser.

## Next Integration Step (Lead Form)

The current form is a placeholder. To make it live:

1. Choose backend destination (Formspree, Zapier, Make, HubSpot, custom API).
2. Replace the submit handler in `script.js` with a `fetch()` call to your endpoint.
3. Add your real contact details in the Contact section.
